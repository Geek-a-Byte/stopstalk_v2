# stopstalk_v2 
- powered by nextjs and flask
- flask api deployed to render
- nextjs frontend deployed to vercel

# run this project

## terminal 1
 - cd Server
 - python -m venv venv
 - cd venv/Scripts
 - activate.bat
 - cd ../..
 - pip install -r requirements.txt
 - py server.py

## terminal 2
 - cd Client
 - npm install next
 - npm run dev
 - open the localhost url
